Веб – приложение для просмотра видео-контента.
Описание: Веб–приложение, где пользователи могут загружать, просматривать и
комментировать видео. Включает категории видео и рекомендации по интересам.
MVP: Регистрация и авторизация пользователей, загрузка видео с выбором категории,
просмотр списка видео с фильтрацией по категориям, отдельная страница просмотра
видео со встроенным плеером, возможность оставлять комментарии под видео (только
для авторизованных пользователей).
Задачи на первый спринт: Задать базовую архитектуру проекта и рабочего процесса.
Реализовать регистрацию и авторизацию пользователя. Создать страницу загрузки видео с
выбором категории (предустановленный список: Спорт, Музыка, Образование, Юмор,
Игры). Настроить сохранение видео на сервер. Создать главную страницу со списком всех
видео (название, категория, автор, дата загрузки). Добавить фильтрацию списка по
категориям. Создать отдельную страницу просмотра видео с HTML5-плеером,
информацией о видео и формой добавления комментария (только для залогиненных
пользователей). Реализовать отображение списка комментариев под видео (имя
пользователя, текст, дата).
